What's Happening

January 2024
What's Happening is an event planner for everyone! Event planners can organize events for a predefined list of venues, invite attendees to your event. Event goers can search for events they want to go to and "reserve" the ones they want to attend. Event search not only searches all the events created within What's Happening, it also searches Ticketmaster using the Ticketmaster Discovery API v2

Technologies & Skills Used: Python, Django, PostgreSQL, CSS (Materialize), API (Ticketmaster Discovery v2), AWS (S3 Bucket), Git, GitHub, Pair Programming

My Flower Garden

December 2023 - January 2024
My Flower Garden will help a user plan and manage the different flowers they plan on planting in the different areas of their yard or garden. It lets you organize by location and container. It also accesses a Plants API from https://perenual.com/ to search for plants or flowers and see some fun facts on those plants.

Technologies & Skills Used: JavaScript, Node.js, Express, EJS, MongoDB, Mongoose, CSS3 (Bootstrap)

What's the Word?

December 2023
What's the Word?! is a version of Wordle that gives you six chances to guess a mystery word. After each guess, you'll find out how close you are - letters in the correct spot will be green, letters in the word but in the wrong spot will be yellow, and letters not in the word at all will be white. The shown keyboard will help you see what letters you haven't yet tried. You can also customize the number of letters in the mystery word - anywhere from 4 - 12!

Technologies & Skills Used: JavaScript, HTML, CSS
